# Disease Outbreak Prediction App

This project is a data science application that analyzes and predicts global disease outbreak trends using time series data.

## 🔗 Live App
https://your-app-link.streamlit.app

## 📊 Features
- Visualizes global confirmed cases
- Moving average smoothing (7-day)
- Time series analysis
- Interactive Streamlit interface

## 🛠 Technologies Used
- Python
- Pandas
- Matplotlib
- Statsmodels
- Streamlit
- Joblib
- Numpy

## 📁 Dataset
Global confirmed cases dataset (CSV format)

## 🚀 How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run app.py
